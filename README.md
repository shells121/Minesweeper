A Minesweeper game built in React. Choose difficulty levels and flag bombs!

## Prerequisites
- Node.js (>=18 recommended)
- npm or yarn

## Installation
```bash
git clone https://github.com/yourusername/minesweeper.git
cd minesweeper
npm install
npm start
